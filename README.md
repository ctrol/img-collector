支持手动采集、自动采集。编辑文章时可一键手动采集；开启自动采集后，保存文章即可自动执行采集操作。

多种采集方式

提供服务器采集、代理采集、浏览器采集三种方式，适配不同服务器环境，解决各类采集问题。

智能图片处理

可自动压缩图片、调整尺寸、转换格式（支持WEBP），支持自定义文件名规则，规范图片管理。

全局扫描采集

可按照文章类型、状态、分类、日期范围等条件批量扫描，一键完成整站外链图片采集。

灵活的过滤规则

可依据顺序、尺寸、格式、域名设置过滤规则，精准筛选需要采集的图片。

图片水印功能

支持文字水印、图片水印，可自定义水印位置、透明度、字体样式，自动为图片添加水印，保护版权。

使用流程

1. 安装激活：上传并激活插件，系统自动创建数据库表及默认配置。
2. 配置设置：根据需求设置采集模式、图片处理规则、过滤条件等参数。
3. 开始采集：编辑文章时点击“保存站外图片”按钮，或启用自动采集模式。
4. 完成本地化：外链图片自动下载至媒体库，文章内图片链接同步替换为本地地址。

常见问题

Q：插件支持哪些图片格式？
A：支持 JPG、JPEG、PNG、GIF、WEBP、BMP、TIFF 等常见格式，也可设置格式过滤规则，排除指定格式。

Q：采集失败怎么办？
A：可切换采集方式，服务器采集失败可尝试代理采集。手动模式下，失败图片会单独列出，支持单张或批量重试。

Q：是否会影响网站性能？
A：插件内置性能优化项，可调整并发数、请求间隔、超时时间等参数，建议在网站访问低峰期执行批量采集。

Q：如何添加水印？
A：在后台水印配置面板开启功能，选择文字/图片水印，并设置位置、透明度等参数；使用该功能需服务器安装 Imagick 扩展。

Q：是否支持古腾堡编辑器？
A：完全支持，同时兼容经典编辑器与古腾堡编辑器。
