# 远程图片采集器 (Img Collector)

一款用于 WordPress 文章远程图片自动采集本地化的插件，支持多种采集模式、过滤规则、水印功能及全局扫描。

[![Version](https://img.shields.io/badge/version-1.8.2-blue.svg)](https://github.com/ctrol/img-collector)
[![License](https://img.shields.io/badge/license-GPL%20v2-blue.svg)](https://www.gnu.org/licenses/gpl-2.0.html)

## 功能特点

### 🎯 核心功能
- **远程图片采集**：自动识别文章中的外链图片并下载到本地
- **微信公众号图片支持**：完美处理微信公众号图片链接，清理域名信息
- **按序替换**：图片按原始顺序替换，保持文章排版不变
- **DOM 属性清洗**：自动清理 img 标签的冗余属性（data-src、data-ratio 等）

### ⚙️ 采集模式
- **手动采集**：点击按钮一键采集当前文章的外链图片
- **自动采集**：保存文章时自动执行采集
- **全局扫描**：批量扫描所有文章，统一处理外链图片

### 🎨 水印功能
- **文字水印**：支持自定义水印文字
- **字体设置**：支持多种字体（Arial、微软雅黑、黑体、宋体等）
- **自定义字体**：可上传 TTF/TTC/OTF 字体文件
- **样式设置**：字体大小、颜色、透明度、位置可调

### ✂️ 过滤规则
- **顺序过滤**：按图片出现顺序过滤
- **尺寸过滤**：按图片尺寸过滤
- **格式过滤**：支持 JPG、PNG、GIF、WebP
- **域名过滤**：白名单/黑名单域名过滤

### 📝 DOM 属性设置
- **Alt 属性处理**：保留原属性、自定义或清空
- **图片对齐**：左对齐、居中、右对齐
- **属性清洗**：自动清理冗余属性（默认启用）

## 安装方法

### 方法一：直接安装
1. 下载 `img-collector-v1.8.2.zip`
2. 进入 WordPress 后台 → 插件 → 安装插件 → 上传插件
3. 上传并激活插件

### 方法二：手动安装
1. 解压 `img-collector-v1.8.2.zip`
2. 将 `img-collector` 目录上传到 `wp-content/plugins/`
3. 进入 WordPress 后台 → 插件 → 激活插件

## 使用说明

### 编辑文章时采集图片
1. 进入文章编辑页面
2. 点击编辑器上方的「采集图片」按钮
3. 在弹出的窗口中选择要采集的图片
4. 点击「开始采集」按钮

### 设置水印
1. 进入「设置」→「远程图片采集器」
2. 切换到「水印设置」标签
3. 启用水印功能
4. 设置水印文字、字体、颜色等参数
5. 保存设置

### 全局扫描
1. 进入「工具」→「远程图片采集器」
2. 设置扫描条件（内容类型、状态、时间范围等）
3. 点击「开始扫描」按钮

## 配置选项

### 基本设置
- **图片质量**：设置采集图片的压缩质量（10-100%）
- **DOM 属性清洗**：启用后清理 img 标签冗余属性（默认启用）
- **图片 alt 属性**：保留原属性、自定义或清空
- **自定义 alt 文本**：自定义 alt 属性的内容
- **图片对齐方式**：左对齐、居中、右对齐

### 水印设置
- **启用水印**：启用文字水印功能
- **水印文字**：自定义水印文字内容
- **字体**：选择水印字体（支持自定义字体）
- **字体大小**：设置水印文字大小（10-100px）
- **字体颜色**：设置水印文字颜色
- **透明度**：设置水印透明度（0-100%）
- **位置**：设置水印位置（9个位置可选）

### 过滤规则
- **启用顺序过滤**：按顺序过滤图片
- **起始位置**：从第几张图片开始采集
- **数量限制**：最大采集数量
- **最小宽度/高度**：过滤尺寸过小的图片

### 代理设置（可选）
- **启用代理**：通过代理服务器采集图片
- **代理主机**：代理服务器地址
- **代理端口**：代理服务器端口

## 系统要求

- WordPress 5.0+
- PHP 7.0+
- GD 库（支持 FreeType）
- 允许远程文件下载（allow_url_fopen）

## 自定义字体

1. 将 TTF/TTC/OTF 字体文件上传到 `wp-content/plugins/img-collector/fonts/` 目录
2. 刷新设置页面，字体选择下拉框会自动显示自定义字体
3. 选择自定义字体即可使用

## 常见问题

### Q: 水印中文显示乱码？
A: 请确保服务器安装了中文字体（如微软雅黑、黑体等），或上传自定义中文字体到 fonts 目录。

### Q: 采集失败？
A: 请检查：
- 服务器是否允许远程文件下载（allow_url_fopen = On）
- 目标图片服务器是否可访问
- 目标图片是否存在

### Q: 水印不显示？
A: 请检查：
- 是否已启用水印功能
- 服务器是否安装了 GD 库和 FreeType
- 水印文字是否为空
